# Movie-Ticket-Pricing-System
Project Title
Movie Ticket Pricing System (Data Analytics Project)

Problem Statement
Movie ticket prices vary based on multiple factors such as:
Movie genre
Show timing
Seat type
Theater location
Demand patterns
Traditional fixed pricing does not maximize revenue or customer satisfaction.
This project aims to analyze pricing patterns and build insights to support data-driven ticket pricing decisions.

Objective
The main goals of this project are:
Analyze historical movie ticket data
Understand factors affecting ticket prices
Tools & Technologies Used

Programming Language: Python
Libraries:
Pandas – Data manipulation
NumPy – Numerical operations
Matplotlib & Seaborn – Data visualization
Environment: Jupyter Notebook

The Movie Ticket Pricing System demonstrates how data analytics can:
Improve pricing strategies
Maximize revenue
Enhance customer satisfaction
